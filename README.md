# K20_HTML-CSS-Group1

Dự án cuối module F8-K20 HTML CSS

# Cáu trúc thư mục 
```text
├─ assets/        // Thư mục chứa các tài nguyên
  ├─ img/               // Thư mục chứa hình ảnh
  ├─ css/               // Thư mục chứa file CSS
  ├─ icon/              // thư mục chứa icon
  ├─ favicon/           // thư mục chứa favicon
├─ pages          // Thư mục chứa các trang HTML
  ├─ about.html          // trang about 
  ├─ projects.html       // trang project 
  ├─ services.html       // trang service
.DS_Store        // file cấu hình 
index.html       // trang chủ (home) 
README.md 
```
